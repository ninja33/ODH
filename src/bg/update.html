在线词典助手-升级说明

您已升级至在线词典助手0.8.3版本。如需完整文档或插件最新信息，请访问 项目主页

0.8.3版新增内容
  1. 修复：在可编辑文本内划词时，偶尔会造成浏览器崩溃，现已禁用在可编辑区域划词(0.8.3)
  2. 改动：移除内置牛津英汉双解词典，留出空间将内置柯林斯英汉双解词条项下的例句补充完整(0.8.2)
  3. 改善：牛津英汉双解在线版改用百度作为数据源，含牛津英汉释义，百度简明释义和百度整句翻译功能。(0.8.2)
  4. 修复：当单词不在以句号，问好，惊叹号结尾的句中，而是单一的单词或在词组中时，不能弹出翻译。(0.8.1)
  5. 新增：为弹窗内原句上下文区域添加富文本编辑功能，当自动选取的句子有问题或希望在句子前后添加注释笔记时可以使用。
  6. 新增：为柯林斯英汉双解词典(有道版)的脚本增加了整句翻译功能，当选取句子而不是单词时，会显示汉语翻译。
  7. 新增：词典单语双语选项(需词典脚本支持)，当该词典脚本支持此功能时，可使弹窗内释义在双语或单语词典之间点击切换(如：英英和英汉词典)
  8. 改善：全面改进了后台选项布局，将词典，anki，脚本选项分开。其中，系统自带脚本将不再以拼接字符串形式显示，而是以列表点选形式激活
  9. 改善：大幅改善在PDF.js所生成的网页中选取句子的准确性。
  10. 新增：新增笔记摘录脚本，当不需要查词典，而仅仅需要摘录网页内容并自行输入笔记时可以使用
0.7版新增内容
  1. 修复ankiconnect制卡超时，造成弹窗内绿色加号变红色叉号，但实际制卡成功的bug(v0.7.1))
  2. 根据广大用户的请求，增加了Ankiweb制卡接口。
  3. 使用Ankiweb接口制卡，可能因违反TOS而被封禁帐号，请谨慎使用，责任自负
  4. 改善使用体验，在输入表单的Text input和Text area内选中单词或按下热键时，将不再弹出翻译，避免在输入文字时反复切换插件开关

本插件主要以脚本获取在线词典信息，下面是部分脚本支持的词典名称，完整内容详见脚本清单

  1. 简明词典：有道简明英汉，百词斩图文词典
  2. 权威词典：柯林斯双解，牛津双解，剑桥双解
  3. 其他词典：法语助手、西语助手、千亿俄汉、汉典汉语
  4. 其他双语：柯林斯英法/法英、剑桥英法/法英词典

提示：云端脚本是指存放在github的最新版脚本，无需等待插件更新而更新。而随插件安装的本地脚本如有bug需等待下一次插件版本才可更新，但加载较快。

注意：ankiconnect官版不支持同一单词重复制卡。想重复制卡，详见这篇博客文章

Online Dictionary Helper -- Upgrade Notes

You extension was upgraded to v0.8.3 Here is the release note. If you need more details, please visit Project Home

version 0.8.3
  1. Bugfix: Sometimes, broswers maybe crash while querying word in editable content. It's been diabled now(0.8.3)

  3. Enhanced: Enhanced Oxford online dictionary wihich uses Baidu as source and integrates Oxfords definition, Baidu concise definition and sentense translaton functions(0.8.2)
  4. BugFix: It will fail to popup definition window when word is not in a sentence(0.8.1)
  5. New: added rich text editor in pop up windows for sentence/context editing
  6. New: added sentence translation function for script encn_Collins(youdao)
  7. New: added dictionary monolingula/bilingual options(need dictionary script support)
  8. Enhanced: re-arranged backend option pages to seperate "dictionary",""anki","script" option tabs. now, users can select system script in list.
  9. Enhanced: improved sentence selection in PDS.JS generated web page.
  10. New: added new "Make notes" script.
version 0.7
  1. fixed bug for ankiconnect 'addNote' timeout(v0.7.1)
  2. added Ankiweb interface based on massive request
  3. Ankiweb account may be disabled according to Ankiweb TOS, please use this function at your own risk
  4. improved user experience while typing text in form fields, like 'input text' and 'text area'

The extension mainly uses script to retrive data from online dictioanry. Below is part of scripts list. For detail, please check Full List

  1. Concise：encn_Youdao, encn_Baicizhan
  2. Learner：encn_Collins, encn_Oxford, encn_Cambridge
  3. Others：frcn_Eudict, escn_Eudict, rucn_Qianyi, cncn_Zdic
  4. Bilingual: Collins(enfr/fren), Cambridge(enfr/fren)
